# socks-proxy
![](https://img.shields.io/badge/node-%3E%3D8.9.1-green.svg)  ![](https://img.shields.io/badge/npm-%3E%3D5.0.0-blue.svg)

使用node实现了基本socks5协议内容, 需要配合对应的客户端使用。

## Installation

```
$ mkdir socks-proxy
$ git clone https://github.com/wleonardo/socks-proxy
```

## Usage

```
// node
node ./demo.js

//or pm2
pm2 start ./demo.js
```
